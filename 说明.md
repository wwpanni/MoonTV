
docker 打包上传

下载

```sh
git clone https://github.com/wwpanni/MoonTV.git
```

打包 

```sh
cd MoonTV
docker build -t junjie11/moontv:latest .
```
登录

```sh
docker login -u junjie11
```

上传

```sh
docker push junjie11/moontv:latest
```
